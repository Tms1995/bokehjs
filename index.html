About bokeh.js

Bokeh is a platform for rich, interactive visualizations targeting the browser. BokehJS is a lower level browser plotting library written in coffeescript, targeted by Bokeh for output.

For more information, please visit the Bokeh web page, or see the Developer Guide to find out more specifically just about bokeh.js.
